Программа для специально обученного сотрудника склада

Алгоритм работы программы:

1. При при запуске локального сервера создается база данных и 3 таблицы:
        `receive` - список поставок на склад
        `dispatch` - список отправок со склада по предзаказам
        `warehouse` - список товаров на складе и их актуальное количество

2. В инпут-поле вводится дата (эмуляция сегодняшнего дня)

3. При сабмите дата уходит на сервер и проходит валидацию

4. В случае ошибок выводится информационное сообщение

5. В соответствии с введенной датой формируется БД:
    каждый день (со дня начала ведения БД по день, введенный в инпут-поле)
    дата записывается в БД с соответствующими ей данными.
    Это эмуляция истории поставок и отправок до сегодняшнего дня

6. При добавлении поставок и отправок в БД, актуальное количество товара на складе пересчитывается автоматически

7. Цена и остаток на складе запрашиваются из БД и отправляются клиенту

8. Данные таблиц удаляются
    (следующая введенная дата в инпут-поле создаст свои данные)

9. Выводится результат в секции с информацией

Для разработки использовался OpenServer
127.0.0.1:3306
